САВЕЛОВ Автоном Петрович — родственник А. Д. Гурьева (см.), непременный член Одесской портовой карантинной конторы, колл. асессор; карточный игрок. Одесский знакомый Пушкина (июнь 1823 — июнь 1824). Из переписки Пушкина с П. А. Вяземским и В. И. Туманским (см.) (янв. — авг. 1825) выяснилось, что С. присвоил («украл») переданные ему Ф. Л. Лучичем для Пушкина 600 руб. (1, 2). «Савелов большой подлец, — писал Пушкин Вяземскому 28 янв. 1825. — Посылаю при сем к нему дружеское письмо» (не сохранилось). Пушкин мог быть знаком с женой С. — Софьей Алексеевной, урожд. Татищевой.

1. Пушкин, XIII, по указ.; 2. Летопись, I, по указ.; 3. ОА, V, вып. 2, с. 125—126; 4. Письма, I, с. 396; 5. Сенатские объявл., 1831, № 7411.
