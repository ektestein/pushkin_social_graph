234

ЛЕОНОВ Степан Алексеевич (1791 — не ранее 1863) — командир Казачьего донского полка, с марта 1828 генерал-майор. По свидетельству И. Т. Радожицкого (см.), Л. присутствовал с Пушкиным 7 июля 1829 на обеде, устроенном И. Ф. Паскевичем по случаю взятия Арзрума (1).

1. Др. и нов. Россия, 1877, № 9, с. 37—38; 2. Лет. ГЛМ, с. 206; 3. Список генералам, 1863, с. 369; 4. Сенатские объявл., 1836, № 1799.
